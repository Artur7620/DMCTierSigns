## ✨ Особенности / Features
- **Unique Icons** — Свои иконки для каждого тира.
- **Smooth Animations** — Плавное появление и исчезновение.
- **Adaptive Opacity** — Иконка тускнеет вплотную.
- **Universal Support** — Работает на любом сервере.
