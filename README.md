# EN
An improved client-side mod for tracking tier signs. This is a fork of DMCTierSigns.

### ✨ Features
- **Unique Icons** for each tier.
- **Smooth Animations** (fade in/out).
- **Universal Support** - works on any server.
- **Cleaner Visuals** - no more clutter.

# RU
Улучшенный клиентский мод для отслеживания тиров на табличках.

### ✨ Особенности
- **Уникальные иконки** для каждого тира.
- **Плавная анимация** появления.
- **Универсальность** - работает на любом сервере.
- **Чистый визуал** - без лишнего мусора.
