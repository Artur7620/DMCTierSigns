# EN

A client-side mod for marking forbidden signs on **dmc-minecraft.net**. Helps quickly find and track tier-marked signs directly in-game.

> **📜 Note:** This mod is made for the **dmc-minecraft.net** server.


### ✨ Features

- **Tier sign detection** - Detects signs with tier text and marks them automatically
- **Through-wall HUD markers** - Displays visible tier markers even when signs are behind walls
- **Tier-based visuals** - Marker color, glow, and particles depend on detected tier
- **In-game config (Mod Menu)** - Adjust marker size, opacity, offsets, and tier visibility


### 📦 Dependencies

- **Fabric API** - Required
- **Mod Menu** - Optional, for in-game configuration GUI

# RU

Клиентский мод для пометок запреток на сервере **dmc-minecraft.net**. Помогает быстро находить и отслеживать таблички с тирами прямо в игре.

> **📜 Примечание:** Этот мод сделан для сервера **dmc-minecraft.net**.


### ✨ Особенности

- **Определение тиров на табличках** - Автоматически находит таблички с текстом тира и помечает их
- **HUD-метки через стены** - Показывает видимые маркеры тиров, даже если табличка за стеной
- **Визуал по тиру** - Цвет, свечение и партиклы зависят от определённого тира
- **Настройки в игре (Mod Menu)** - Размер, прозрачность, смещение и фильтр по тирам


### 📦 Зависимости

- **Fabric API** - Обязательно
- **Mod Menu** - Опционально, для GUI настроек в игре
